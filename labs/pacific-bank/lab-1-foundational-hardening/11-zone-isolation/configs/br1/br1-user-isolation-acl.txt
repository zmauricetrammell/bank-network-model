! ACL for BR1 user LAN
ip access-list extended BR1_USER_IN
    remark 0) Allow users access to gateway
    permit ip 10.1.10.0 0.0.0.255 host 10.1.10.1

    remark 1) Allow users all services to corporate resolver
    permit ip 10.1.10.0 0.0.0.255 host 10.0.10.100

    remark 2) Allow users "internet" access
    permit ip 10.1.10.0 0.0.0.255 203.0.113.0 0.0.0.255

    remark 3) Deny users to all corporate private networks
    deny ip 10.1.10.0 0.0.0.255 10.0.0.0 0.255.255.255

    remark 4) Deny users to WAN networks 
    deny ip 10.1.10.0 0.0.0.255 172.16.0.0 0.15.255.255

    remark 5) Default action (lab friendly)
    permit ip 10.1.10.0 0.0.0.255 any

! Add to interface
interface FastEthernet1/0
    ip access-group BR1_USER_IN in
